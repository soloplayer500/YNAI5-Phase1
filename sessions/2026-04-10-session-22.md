### 3. GDrive FUSE Hang — Root Cause Fix
**Problem:** `/api/status`, `/status.json` timing out (never returned). Dashboard HTML served fine but all API endpoints hung.

**Root cause:** `main.py` had two blocking calls on the hung rclone FUSE mount:
1. `drive_mounted = os.path.exists(DRIVE_ROOT) and bool(os.listdir(DRIVE_ROOT))` — `os.listdir()` on a FUSE mount hangs if FUSE is unresponsive
2. `open(HEARTBEAT_PATH)` in `read_heartbeat()` — opens a file inside `/mnt/gdrive/` which hangs

**Fix attempt 1:** `is_drive_mounted()` using `subprocess.run(['mountpoint', '-q', ...])` — ALSO hangs because `mountpoint` calls `stat()` on the path.

**Fix attempt 2 (final):** Read `/proc/mounts` directly — pure kernel file, zero FUSE access:
```python
def is_drive_mounted() -> bool:
    try:
        with open('/proc/mounts') as f:
            return any(DRIVE_ROOT in line for line in f)
    except Exception:
        return False
```
Guard `read_heartbeat()` behind `is_drive_mounted()` before touching any file path inside `/mnt/gdrive`.

**Result:** All endpoints 200 OK with fast response times.
