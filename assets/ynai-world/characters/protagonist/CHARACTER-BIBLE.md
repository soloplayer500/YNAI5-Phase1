## How to Use These in Video Generation

### Veo 3.1 — "Ingredients-to-Video" (AI Studio UI)
1. Go to: https://aistudio.google.com/models/veo-3
2. Click "Ingredients" / reference image upload
3. Upload **ref-06** as primary + **ref-02** as secondary
4. Write prompt describing the scene, NOT the character (Veo extracts character from refs)
5. Veo maintains character consistency across the generated clip

**Example prompt with refs:**
```
The character stands in a dark alley at midnight. Neon purple light reflects off wet pavement.
Floating skulls circle slowly behind him. He looks directly into the camera.
Atmosphere: cinematic, ominous, powerful. 9:16 vertical portrait format.
Native audio: distant city sounds, low electronic hum, wind.
```

### Sora 2 — Image-to-Video (ChatGPT Pro)
1. Upload **ref-02** or **ref-06** as the reference image
2. Describe the motion/scene you want to add
3. Sora uses the image as style/character reference — not locked as first frame

**Example prompt with ref:**
```
Reference image style. The character slowly raises his head, eyes glowing brighter.
Purple smoke begins swirling around him. Subtle, slow motion. Dark background.
```
