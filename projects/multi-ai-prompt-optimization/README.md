# Multi-AI Prompt Optimization
**Status:** 🔨 Active Development
**Type:** Prompt engineering framework for AI models

---

## What It Is
A system for designing and optimizing prompts for different AI models. Different AI models require different prompt structures to perform well — this project documents and refines those frameworks.

## Problem It Solves
- Eliminates trial-and-error with AI prompts
- Creates reusable, reliable prompt templates
- Builds a personal library of tested prompt patterns
- Documents model-specific behaviors and best practices

## Frameworks

| Framework | File | Target Model |
|-----------|------|--------------|
| CPO (Claude Prompt Optimizer) | cpo-framework.md | Claude |
| LYNX-K | lynx-k-framework.md | Kimi |
| Templates | templates/ | Various |

## Current Focus
Documenting CPO framework based on learnings from Claude usage.

## Principles
- Treat prompts like structured programs, not simple instructions
- Include guardrails and constraints explicitly
- Model-specific optimization beats generic prompting
- Document what works, discard what doesn't
