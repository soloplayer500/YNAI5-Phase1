# DISTRIBUTION — Platform Specs & Formatting Rules

## Twitter / X
- **Char limit:** 280 per tweet
- **Thread:** Chain tweets with 1/ 2/ 3/ numbering
- **Hashtags:** 3-5 max, place at end
- **Links:** Count toward 280 chars (Twitter shortens to ~23 chars)
- **Images:** Up to 4 per tweet
- **Best posting times:** 8-10am, 12-1pm, 5-6pm EST
- **YNAI5 tone:** Direct, opinion-forward, punchy. No corporate speak.
- **Format template:**
  ```
  [Hook statement — 1 line]
  
  [1-3 supporting points]
  
  [CTA or closing thought]
  
  #Hashtag1 #Hashtag2 #Hashtag3
  ```

## Telegram (Block Syndicate / Openclaw)
- **Char limit:** 4096 per message
- **Formatting:** Markdown supported — **bold**, _italic_, `code`, [link](url)
- **Signals:** Use bold for key numbers, emoji for visual scanning
- **Pinning:** Pin important signals so they stay visible
- **Buttons:** Use inline keyboards for interactive signals (Buy/Sell/Skip)
- **Channels vs Groups:** Channels = broadcast only. Groups = discussion.
- **Block Syndicate public:** condensed signal, no detailed thesis
- **Block Syndicate VIP:** full signal with thesis, entry details, risk breakdown

## TikTok
- **Caption limit:** 2200 chars (but 150 ideal before fold)
- **Hashtags:** 5-8 in caption, niche + broad mix
- **No clickable links in caption** — link in bio only
- **Tone:** Casual, authentic, direct. No brand-speak.
- **Hook rule:** First line must grab in <3 seconds
- **OpenMind AI hashtags:** #AI #ArtificialIntelligence #OpenMindAI #AINews #Tech #FutureOfAI

## Instagram
- **Caption limit:** 2200 chars
- **Hashtags:** 20-30 in first comment (not caption) for best reach
- **First 125 chars:** Must hook before "more" truncation
- **Stories:** 15 sec video, swipe-up only with 10k+ followers
- **Reels:** Caption same rules as feed post
- **YNAI5 hashtags:** #YNAI5 #AI #ArtificialIntelligence #Crypto #MusicProduction #AIMusic

## LinkedIn
- **Post limit:** 3000 chars
- **Hashtags:** 3-5 max, at very end
- **Tone:** Professional but authentic — not stiff
- **Line breaks:** Use frequent line breaks — LinkedIn compresses walls of text
- **Best for:** AI industry takes, YNAI5 brand building, thought leadership
- **No:** Slang, excessive emoji, crypto price speculation

## YouTube (Community Posts)
- **Char limit:** None practically
- **Best for:** Announcements, behind-the-scenes, link shares
- **Tone:** Casual, like talking to subscribers
- **Video descriptions:** 500 chars before fold, include timestamps

## Reddit
- **Use for:** Research monitoring only via PLAYWRIGHT
- **Not for:** Direct posting (requires account auth and community rules vary)
- **Best subreddits to monitor:** r/artificial, r/MachineLearning, r/CryptoCurrency, r/Bitcoin

## Blotato (Multi-platform content repurposing)
- **What it does:** Input one piece of content → repurpose for all platforms automatically
- **Connect to:** YNAI5 Blotato account
- **Best for:** Symphony music posts, YNAI5 brand content that needs to go everywhere at once
- **Use via:** PLAYWRIGHT or native Blotato web interface
