# Client Services — Mission

## North Star
Generate predictable per-week cash flow from automation gigs while building a portfolio of
reusable productized services that can scale via templating (each new client costs less time
than the last).

## Why It Exists
1. **Revenue floor:** Gives the operator a non-zero income while crypto-alpha lane builds MRR
2. **Reusability test:** Forces existing scripts to be cleanly packaged, documented, and reusable
3. **Reputation:** Reviews on Fiverr → portfolio → higher-ticket direct work later

## Three Outcomes (ordered)
1. **First $50 order** within 14 days
2. **5 completed orders** (5-star reviews) within 60 days → unlock raised pricing
3. **$500/mo cumulative** within 90 days from gigs alone

## Operating Principles
- **Overdeliver on first 5 orders.** Reviews compound; pricing can be raised after 5 stars stack.
- **Productize, don't customize.** Each gig has a fixed deliverable. No scope creep without upcharge.
- **Templated everything.** README + setup video + handoff checklist that scales.
- **Same-hour first response.** Fiverr ranking algorithm rewards velocity.
- **Never deliver work the operator can't maintain post-handoff** (avoid one-off custom forks).

## Non-Negotiables
- No financial advice gigs. Tools only, never picks.
- No credentials handling — clients manage their own bot tokens, API keys
- No 24/7 hosting commitments unless tier explicitly includes it
- No refunds after delivery accepted (Fiverr standard) — be ruthless about quality before sending
