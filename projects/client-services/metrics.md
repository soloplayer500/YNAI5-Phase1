## Quality Metrics (per order)
| Metric | Target |
|--------|--------|
| Time on delivery (Basic) | ≤4 hrs |
| Time on delivery (Standard) | ≤8 hrs |
| Time on delivery (Premium) | ≤16 hrs |
| Effective hourly rate | ≥$25/hr (Basic), ≥$30/hr (Premium) |
| Revisions per order | ≤1 |

## Key Ratio
**Revenue per delivery hour** is the only metric that matters at scale.
If Basic orders drop below $20/hr → either raise price or productize harder.
