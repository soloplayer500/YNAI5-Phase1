# Ledger Node
**Role:** Financial reasoning and risk evaluation

---

## Purpose
The Ledger Node handles all financial analysis, trade evaluation, and risk assessment. It separates emotional financial decisions from structured, evidence-based analysis.

## Responsibilities
- Evaluate financial decisions with structured analysis
- Assess risk vs reward for trades and investments
- Separate emotional signals from objective data
- Apply predefined rules and criteria
- Output clear go/no-go recommendations with reasoning

## When to Use
- Any crypto or stock trade decision
- Budget allocation decisions
- Evaluating new tools or services with costs
- Assessing financial risk of any decision

```
Asset: [ticker/instrument]
Current Price: [price]

```
