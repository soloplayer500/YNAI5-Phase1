# Future Ideas — Metrics

## The only metric: count, not value

| Metric | Target | Why |
|--------|--------|-----|
| Total parked items | <15 | Anything more = backlog rot, not parking |
| Items reviewed last quarter | 100% | Untouched parking = same as deletion |
| Promotions to active lane / quarter | ≤1 | Discipline gate. More = focus dilution. |
| Archives / quarter | 0–3 | Healthy pruning, not panic deletion |
| New entries / month | ≤2 | Cap input velocity to keep lane sane |

## Anti-Metrics (signals of failure)
- Reviewing this folder mid-session during active-lane work → focus drift
- Creating tasks in tasks.md outside quarterly review → premature execution
- Promoting >1 idea/quarter → focus dilution
- Items aged >365 days unreviewed → archive automatically next review
