# Future Ideas — Agents

_No active agents. Lane is dormant by design._

## Manager Agent
**Identity:** Operator (Shami) — quarterly only
**Role:** Reviews each parked item, decides: kill / promote / re-park
**Cadence:** First week of each quarter (Q1, Q2, Q3, Q4)
**Boundaries:** Does NOT execute anything. Reviews and decides.

## Worker Agent
**Status:** None. Future-ideas does no work.

## Research Agent
**Identity:** Brave Search MCP (used only during quarterly review)
**Role:** Quick scan of "is this idea still relevant?" — market changes, competitor shifts
**Cadence:** Quarterly only, ~15 min/idea max
**Boundaries:** Read-only. Outputs a one-line recommendation.

## Decision: Lane is intentionally agent-light
The whole point is to NOT spend execution cycles here. Adding agents creates pressure to "use them"
which violates the parking-lot mission.
