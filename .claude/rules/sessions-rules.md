# Session File Rules

When creating or editing files in the `sessions/` directory:

- Filename format: `YYYY-MM-DD-session.md` (regular) or `YYYY-MM-DD-weekly-review.md` (weekly)
- Always include all template sections: Focus, What Got Done, Decisions Made, Open Items, Next Steps, Memory Updates
- Open Items must use checkbox format: `- [ ] item`
- Memory Updates section is required — do not leave blank, write "none" if nothing to record
- Do not delete existing session files — they are the permanent record
- After creating, update `memory/MEMORY.md` Session Index
